Output XML Schema

The Output XML schema is an XML schema for communicating the output from an execution of IBM® SPSS® Statistics to another application. Its emphasis is on content, not presentation, and on accessibility to general tools like XSLT rather than on compactness.

The schema covers nearly all the object types that can be in output, including:

• Text blocks (including logs).

• Pivot tables (including notes and warnings tables).

• "Standard" charts (charts not created by the IGRAPH command).

• Headings that organize output in the outline pane of the Viewer.
