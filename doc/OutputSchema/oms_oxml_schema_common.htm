Common Attributes

Unlike simple text output, where what you see in the Viewer is basically what you get in the OXML <textBlock> element, the OXML <pivotTable> element can contain much more information than simply the text displayed in the pivot table. In addition to the text attribute that contains the text displayed in the pivot table, the child and descendant elements of the <pivotTable> element can contain additional attributes that describe the data used to construct that text.

Numbers

Elements that contain numbers that are not values of IBM® SPSS® Statistics variables can have the following attributes:

number. The numeric value, up to 14 significant digits.

format. The SPSS Statistics format used to display the number. For the default F format, the format attribute is not included in the element.

decimals. The number of decimal positions displayed.

null. Where a statistic cannot be computed, the Viewer displays the system-missing value: a decimal point (.). This is represented by null='true' in OXML. None of the other attributes usually shown with numbers are shown for null values.

Dates, Times, Time Intervals

Dates, times, and time intervals are treated as special kinds of numbers and are represented by different attributes:

date. Attribute values are xs:dateTime values of the general form yyyy-mm-ddThh:mm:ss.ss. The format attribute value indicates the SPSS Statistics date format. For example: 

<cell date="2003-08-05T17:44:04.48" format="DATETIME" text="05-AUG-2003 17:44:04" />

time. Attribute values are xs:time values of the general form hh:mm:ss.ss. The format attribute value is usually time.

duration. Attribute values are xs:duration values of the general format PnYnDTnHnMn.nS. The format attribute value can be either time or dtime. For example:

<cell duration="P10DT11H12M00S" format="DTIME" text="10 11:12" />

As with other numbers, if the value is the system-missing value, the element will have the null="true" attribute.

Numeric Data Values

The values of numeric variables are, of course, numbers, and they can have the same attributes as other numbers. Their values will be represented by a number, date, time, or duration attribute, depending on the variable's format, unless the value is system-missing. The system-missing value is represented by null="true".

Elements that represent values of SPSS Statistics variables can also have the following attributes (in addition to the text attribute):

varName. Attribute values are case-sensitive variable names.

label. Attribute values are the defined value labels. If the value doesn't have a defined label, the label attribute is not included.

display. Attribute values are the display rules for the values: value, label, or both. The display attribute is only included if it differs from the valueDisplay attribute of the ancestor <pivotTable> element.

Text Strings

The values of simple text strings like statistic names, and the values of composed strings like most footnotes, appear in the XML with only the text attribute, whose value is the string itself.

String Data Values

In addition to the text attribute, values of string variables have the following attributes:

string. Attribute values are xs:string values that contain the actual values of the string variables.

varName. Attribute values are case-sensitive variable names.

label. Attribute values are the defined value labels. If the value doesn't have a defined label, the label attribute is not included.

display. Attribute values are the display rules for the values: value, label, or both. The display attribute is only included if it differs from the valueDisplay attribute of the ancestor <pivotTable> element.

Variables

SPSS Statistics variables are described by the following attributes:

varName. Attribute values are case-sensitive variable names.

label. Attribute values are the defined variable labels. If the variable doesn't have a defined label, the label attribute is not included.

variable. The attribute value of variable="true" distinguishes variables from data values.

display. Attribute values are the display rules for the variables: name, label, or both. The display attribute is only included if it differs from the tableVariableDisplay or outlineVariableDisplay attribute of the ancestor <command> element.
