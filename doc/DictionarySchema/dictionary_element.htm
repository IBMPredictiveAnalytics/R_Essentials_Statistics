dictionary Element

<dictionary> is the root element of the schema. In addition to namespace declaration and schema location information, it has three attributes:

• locale. Locale in ISO format. The language part is ISO 639 and the country part is ISO 3166.

• creationDateTime. Date-time in ISO 8601 format.

• rowCount. Number of rows in the active dataset. This value is not affected by weighting or filtering.

Example

<dictionary locale="en_US" creationDateTime="2002-08-26T14:49:24"
  rowCount="200" xmlns="http://xml.spss.com/spss/data" 
  xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" 
  xsi:schemaLocation="http://xml.spss.com/spss/data 
  http://xml.spss.com/spss/data/dictionary-1.0.xsd">

Child Elements

The <dictionary> element has the following child elements:

• <variable>

• <valueLabelSet>

• <weightVariable>

• <multipleResponseSet>

• <variableSet>

• <attributeSet>
