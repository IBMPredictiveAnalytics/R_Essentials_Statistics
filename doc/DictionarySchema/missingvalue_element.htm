missingValue Element

The <missingValue/> element contains user-missing value definitions. A <variable> element can contain up to three <missingValue/> elements. If a variable has no user-missing values, this element is not present.

Attributes

• data. Data value that represents a defined missing value or beginning or end of a range of missing values. The value can be any valid value for the variable type or "lowest" or "highest" for missing value ranges for numeric variables.

• type. For values that represent the start or end of missing value ranges, the type attribute has a value of "lowerBound" or "upperBound". For discrete missing values, the type attribute is not present.

Example

<missingValue data="8"/>
<missingValue data="9" type="lowerBound"/> 
<missingValue data="highest" type="upperBound"/>
