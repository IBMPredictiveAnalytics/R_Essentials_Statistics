valueLabel Element

The <valueLabel/> element contains information for a single value-label pair. A <valueLabelSet> element contains one or more <valueLabel/> elements.

Attributes

• label. A descriptive value label. A string up to 120 bytes long.

• value. The data value associated with the label.

Example

<valueLabel label="Low" value="1" />

See the topic valueLabelSet Element for more information.
