valueLabelVariable Element

The <valueLabelVariable/> element identifies the variable associated with a set of value labels. A <valueLabelSet> element contains one <valueLabelVariable/> element.

The <valueLabelVariable/> element has a single attribute, name, that contains the name of the variable associated with the value label set.

Example

<valueLabelVariable name="numvar1" />

See the topic valueLabelSet Element for more information.
