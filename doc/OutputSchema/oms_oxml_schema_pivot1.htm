Pivot Table Child Elements

The <pivotTable> element always has the following child element:

dimension. The first dimension in the table and the root of the subtree that contains the body of the table.

The <pivotTable> element can also have one or more of the following additional child elements:

footnote. Parent of the <note> element that contains footnote text. The <footnote> element has no attributes; it contains or more child <note> elements.

caption. Text displayed below the pivot table. This may have any of the common text attributes.

cornerText. Text displayed in the upper left corner of the pivot table. This may have any of the common text attributes.
