variable Element

Most of the metadata attributes associated with individual variables are contained in attributes and child elements of the <variable> element.

Attributes

• alignment. Alignment of data values in the Data Editor. The value can be "left, "right", or "center".

• displayWidth. Column width for the display of variables in the Data Editor. The value is a positive integer.

• label. Descriptive variable label. The value is a string with a maximum length of 255 bytes. If the variable doesn't have a variable label, this attribute is not present.

• measurementLevel. Measurement level. The value can be "nominal", "ordinal", "scale", or "unknown".

• name. Variable name. The value is a string with a maximum width of 64 bytes.

• type. Variable type, numeric or string. The value is 0 for numeric variables or a positive integer indicating the defined width (in bytes) for string variables.

Example

<variable alignment="right" displayWidth="11" 
  label="Descriptive variable label" measurementLevel="ordinal" 
  name="numvar1" type="0">
