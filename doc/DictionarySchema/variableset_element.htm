variableSet Element

The <variableSet> element contains information on defined variable sets (used to control the variables displayed in dialog selection lists). The <dictionary> element can contain one or more <variableSet> elements. If there are no variable sets in the active dataset, this element is not present.

The <variableSet> element has one attribute, name, that contains the name of the variable set.

Child Elements

The <variableSet> element contains one or more <variableSetVariable/> elements that identify the variables in the set. The <variableSetVariable/> element has a single attribute, name, that contains the name of a variable in the set.

Example

<variableSet name="SetOne">
  <variableSetVariable name="numvar1" /> 
  <variableSetVariable name="numvar2" /> 
</variableSet>
