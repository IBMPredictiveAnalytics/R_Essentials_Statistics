multipleResponseSetDichtomy Element

The <multipleResponseSetDichotomy/> element identifies the counted value for multiple dichotomy sets.

• The <multipleResponseSet> element can contain one <multipleResponseSetDichotomy/> element.

• If this element is present, the <multipleResponseSetCategoryLabels/> element must also be present in the <multipleResponseSet> element.

• The <multipleResponseSetDichotomy/> element has a single attribute, value, that identifies the counted value.

• For multiple category sets, this element is not present.

Example

<multipleResponseSetDichotomy value="1" />

See the topic multipleResponseSet Element for more information.
