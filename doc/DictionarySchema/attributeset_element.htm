attributeSet Element

The <attributeSet> element contains information on variable and/or file attributes (typically defined with the VARIABLE ATTRIBUTE and DATAFILE ATTRIBUTE commands).

• <dictionary> and <variable> elements can contain one or more optional <attributeSet> elements.

• If the element is a child of the <dictionary> element, it is a file attribute set.

• If the element is a child of the <variable> element, it is a variable attribute set.

• <attributeSet> has one attribute, version, that identifies the version number of the set (how many times the set has been modified).

Child Attributes

• The <attributeSet> element has single child element, <attribute>, which in turn has a single child element <attributeValue/>.

• <attributeSet> contains one or more <attribute> elements.

• <attribute> has two attributes: name contains the name of the attribute, and type identifies the type as "user" or "system".

• <attribute> contains one <attributeValue/> element, with one attribute, value, that contains the value of the attribute.

Example

<attributeSet version="1">
  <attribute name="dataAttribute1" type="user">
    <attributeValue value="A datafile attribute" /> 
  </attribute>
</attributeSet>
