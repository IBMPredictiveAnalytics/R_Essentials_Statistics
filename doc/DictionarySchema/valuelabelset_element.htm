valueLabelSet Element

The <valueLabelSet> element contains one or more <valueLabel/> elements and one <valueLabelVariable/> element that contain information on defined value labels. The <dictionary> element can contain one or more <valueLabelSet> elements. There is a separate <valueLabelSet> element for each variable that has defined value labels. If no variables in the active dataset have defined value labels, this element is not present.

The <valueLabelSet> element has no attributes.

Example

<valueLabelSet>
  <valueLabel label="Low" value="1" /> 
  <valueLabel label="Medium" value="2" /> 
  <valueLabel label="High" value="3" /> 
  <valueLabelVariable name="numvar1" /> 
</valueLabelSet>

Child Element

• <valueLabel/>

• <valueLabelVariable/>
